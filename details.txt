# 🛍️ Walmart Sales Data Analysis & Forecasting

A Streamlit-powered data science dashboard that analyzes Walmart’s historical weekly sales data, performs forecasting using machine learning models, detects anomalies, and allows sales prediction through user input.

---

## 📌 Project Overview

This project uses Walmart's weekly sales dataset to provide:

* 📊 **Sales Insights & Visualizations** (Store-wise, Time-series)
* 🔍 **Missing Value & Anomaly Detection**
* 📈 **Machine Learning Models for Forecasting**
* 🔮 **Live Sales Prediction using User Input**
* 📦 **Inventory Forecasting with XGBoost**
* 🎉 **Holiday Impact Analysis**
* 📄 **PDF Report Generation**
* 🌐 **Deployed via Streamlit Cloud**

---

## ⚙️ Technologies Used

| Category          | Tools / Libraries                     |
| ----------------- | ------------------------------------- |
| Web UI            | Streamlit                             |
| Data Handling     | pandas                                |
| Visualization     | plotly, seaborn, matplotlib           |
| Machine Learning  | scikit-learn (Random Forest), XGBoost |

## 🔧 Key Features

### 🗃️ Data Upload

Upload your own CSV file of Walmart sales data and the app will begin analysis automatically.

### 📊 Exploratory Data Analysis

* Dataset preview and summary
* Missing value detection
* Sales by store
* Sales trend over time

### 💡 Machine Learning Models

* **Random Forest**: For weekly sales prediction based on store, temperature, CPI, fuel price, unemployment, and holiday flag.
* **XGBoost**: For inventory forecasting based on same features.
* **Prophet**: For time-series forecasting of future sales.

### 🚨 Anomaly Detection

Uses Isolation Forest to highlight unusual weekly sales spikes/dips.

### 🔮 Predict Weekly Sales (Interactive)

Users can input values for:

* Store number
* Temperature
* CPI
* Fuel Price
* Unemployment Rate
* Holiday or Not
  ➡️ And get real-time sales prediction!

### 📄 PDF Report Export

Auto-generates a clean PDF summarizing:

* Key prediction result
* Models used
* Explanation of input features

---

## 🧪 Sample Input Features for Prediction

* **Store**: Store number (1–50)
* **Temperature**: In Fahrenheit
* **Fuel\_Price**: Fuel cost per gallon
* **CPI**: Consumer Price Index
* **Unemployment**: In percentage
* **Holiday**: Yes/No flag
